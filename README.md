# todo-cli-app

A simple command-line To-Do list app written in Python.

 

## Features

- Add new tasks

- View all tasks

- Mark tasks as completed

- Saves tasks to a file (`tasks.txt`)

- Uses Object-Oriented Programming

 

## How to Run

 

1. Make sure you have Python installed.

2. Download `todo_cli_app.py`.

3. Open a terminal or command prompt.

4. Run the file with:

 

```

python todo_cli_app.py

```

 

5. Follow the menu to manage your tasks!

 

## Example

 

```

--- To-Do List ---

1. Add Task

2. View Tasks

3. Mark Task as Done

4. Exit

```
